# USDT HUB

> 一个持续更新的公开笔记。写给第一次接触 USDT、TRON、钱包和链上支付的人。

围绕 `Get / Transfer / Earn / Spend / Tools` 五个任务入口组织内容。

这不是行情站，也不是平台导购页。你如果只是想先把第一笔 USDT 用明白，这里够用。

我主要盯三件事：

- 怎么买、怎么存、怎么转，尽量别在第一步就踩坑。
- TRC20、手续费、TRX、能量这些最容易卡住人的地方，直接拆开讲清楚。
- 等你真拿到 USDT 以后，它除了放着，还能怎么用，比如 AI、旅游、eSIM、VPN 和跨境支付。

> 先把丑话说前面。这不是交易平台，也不是投资建议。我写这些，不是为了劝你频繁操作，而是为了让你第一次动手时少交学费。这个领域里，最贵的通常不是那几块手续费，而是地址填错、网络选错、控制权没搞明白。

## 3. 默认起步路径

这里不做“最强工具榜”，只给一套对新手更稳的起步路径。

| 你现在的目标 | 我会建议先看什么 | 原因 |
| --- | --- | --- |
| 想先买到第一笔 USDT | [如何购买 Tether USDt（USDT）](./markdown/how-to-buy-usdt.md) | 先把买法和风险路径跑通 |
| 买完以后不知道放哪 | [怎么选钱包](./markdown/how-to-choose-a-wallet.md) | 平台账户和自托管不是一回事 |
| 准备第一次转账 | [怎么发送 USDT](./markdown/how-to-send-usdt.md) | 绝大多数事故都发生在这一步 |
| 总被 TRC20 / ERC20 搞混 | [TRC20、ERC20、BEP20 有什么区别](./markdown/usdt-networks-explained.md) | 错链是最常见的低级事故 |
| 想知道哪条链现在转账更省 | [USDT 不同链转账手续费对比](./markdown/usdt-gas-fee-comparison.md) | 先把成本带和错误直觉纠正过来 |
| 想算质押多少 TRX 才够发 | [TRON 能量计算器](./markdown/tron-energy-calculator.md) | 直接算能量、笔数和差额，少靠脑补 |

如果你现在就要开始管理 TRON 和 USDT，我通常会建议先用 [imToken](https://token.im/trx-wallet) 起步。不是因为它“对所有人都最好”，而是因为它对新手最关键的几件事做得相对集中：创建自托管钱包、备份助记词、查看 TRON 资源、处理 TRC20 USDT 收发，以及在需要时直接租能量，把整套流程先跑通。

## 5. 部署和维护

这个仓库的源文件在 `site/`，构建产物在 `docs/`，GitHub Pages 直接服务 `docs/`。

部署、预览、验证、发布的完整说明，单独放在这里：

- [DEPLOY.md](./DEPLOY.md)

一句话版：

```bash
bundle exec jekyll build
python3 scripts/check_site.py
python3 scripts/test_usdt_tools.py
./scripts/deploy.sh "commit message"
```

注意：
- 本地预览优先用 `bundle exec jekyll serve`
- 不要直接裸跑 `python3 -m http.server docs`
- 这个站有 `baseurl: /usdthub`，预览姿势错了，看起来就会像站坏了，其实只是路径炸了
