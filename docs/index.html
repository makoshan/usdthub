USDT 新手手册

一个持续更新的公开笔记。写给第一次接触 USDT、TRON、钱包和链上支付的人。

我做这个手册，只想解决两类问题：

  • USDT 怎么买、怎么存、怎么转、怎么避坑。
  • USDT 除了放着，还能拿来做什么，比如 AI、旅游、eSIM、VPN 和跨境支付。
这不是交易平台，也不是投资建议。
我更关心的是：如果你第一次实际使用 USDT，应该先理解什么，先准备什么，哪里最容易翻车。很多站喜欢把你往开户链接和工具页推，我更想先把路讲明白。因为在这个领域里，真正贵的是你第一次就走错。

2. 常用外部 App 与工具

这里不做「最强工具榜」，只给一套真的适合第一次上手 TRON / TRC20 USDT 的起步工具。顺序也很重要：先会用钱包，再学会查区块浏览器，最后才去碰 DEX、借贷和跨链。

默认起步方案

如果你现在就要开始管理 TRON 和 USDT，我通常会建议先用 imToken 起步；如果你后面准备更深地用 TRON 生态，再补一个 TronLink。前者对新手更友好，后者更像 TRON 原生工具箱。无论你选哪个，先把「备份助记词、收一笔小额 TRX、收一笔小额 USDT、查一次交易记录、做一次小额转账」这五步跑通，比上来就研究 DeFi 更重要。

工具类型 新手先用哪个 主要用来干什么
钱包 imToken / TronLink 创建自托管钱包、收发 TRX / TRC20 USDT、查看资源
区块浏览器 TRONSCAN 查地址、查交易、看带宽 / 能量、确认 USDT 是否到账
费用比较 USDT 不同链转账手续费对比 先看哪条链的链上成本更低，别靠印象选
TRON 估算 TRON 能量计算器 直接算质押多少 TRX、够转几笔、不够差多少
DEX 兑换 SUN.io 在 TRON 上兑换代币，查看池子、路由和价格
跨链 BTTC Bridge + BTTCScan 在 TRON / Ethereum / BSC 之间转资产，跨链后查记录
