# Homepage Information Architecture

Page goal
- Help first-time visitors understand what the site is for
- Route them into the correct learning path
- Build trust before introducing tool recommendations
- Make imToken appear as a beginner-friendly option inside workflow context, not as the homepage's main pitch

Primary CTA
- Chinese: 从这里开始学习 USDT
- English: Start Learning USDT

Secondary CTA
- Chinese: 查看专题矩阵
- English: Browse Topics

## Recommended homepage module order

### 1. Hero
Headline
- 中文: USDT 新手指南，教你买、存、转、用
- 英文: Learn how to buy, store, send, and use USDT

Support copy
- Explain two question types:
  1) how to use USDT
  2) what you can do with USDT

Trust line
- Third-party guide for beginners
- Not a trading platform
- Not investment advice

CTA
- Primary: Start Learning USDT
- Secondary: Browse Topics

### 2. 4 entry cards
- 买 USDT / Buy USDT
- 选钱包 / Choose a Wallet
- TRON / TRC20 转账 / TRON Transfers
- USDT 能拿来做什么 / Use Cases

Each card should answer:
- What problem it solves
- Who it is for
- Where to click next

### 3. Beginner path
Suggested steps
1. 了解 USDT / Understand USDT
2. 选钱包 / Choose a wallet
3. 买入与转账 / Buy and send
4. 节省手续费 / Reduce fees
5. 开始消费 / Start using it

### 4. Content centers
- Learn
- TRON / Fees
- Use Cases
- Compare
- Tools / Explainers

### 5. Editor's pick / beginner-friendly tools
Purpose
- Soft conversion block
- Explain why a tool is easier for beginners
- Avoid brand-first language

Suggested card structure
- Best for beginners: imToken
- Why:
  - easier self-custody onboarding
  - clear TRON / USDT handling
  - suitable for first-time on-chain users

Also show 1-2 alternatives with “best for” framing

### 6. Popular articles
Suggested first set
- USDT 是什么？/ What Is USDT?
- 怎么买 USDT？/ How to Buy USDT
- 钱包怎么选？/ How to Choose a USDT Wallet
- 为什么转 USDT 要 TRX？/ Why Do You Need TRX?
- 什么是 TRON 能量？/ What Is TRON Energy?
- USDT 可以做什么？/ What Can You Do With USDT?

### 7. FAQ
Core questions
- What is USDT?
- How do beginners buy USDT?
- Which wallet is easier for beginners?
- Why do I need TRX to send USDT on TRON?
- What is TRON energy?
- What can you pay for with USDT?

### 8. Editorial trust block
Include
- How we review wallets and apps
- Editorial policy / disclosure
- Risk statement
- Regional availability disclaimer

## Copy rules
- Tutorial tone, not ad tone
- Explain before recommending
- “Best for beginners” > “best wallet ever”
- “Editor’s pick” > “download now”
- Always include risk / legality / availability disclaimers where relevant

## Conversion rules
- Homepage should not lead with wallet download
- Wallet recommendation appears after context and explanation
- Stronger product recommendation can appear after article 2 or 3 in the user journey
