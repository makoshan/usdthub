# C2C / OTC 到底在怕什么

> China Notes · Payment Risks

新手第一次听到 C2C / OTC，重点常常放在“汇率”“点差”“谁更便宜”。大陆老用户紧张的往往不是价格，是后面那条支付链条和事后解释的成本。

> TL;DR C2C / OTC 怕的基本不是“买不到”。真正的麻烦在交易对手、支付工具风控、资金来源、账户冻结、事后解释，以及你自己判断不了这笔交易干不干净。

## 1. 交易对手风险


**交易对手风险**指的是：C2C / OTC 上你看到的只是报价、昵称和完成率，对对方的真实身份、资金来源、是否正在被其他案件追查一无所知。只要对方链条有问题，麻烦就可能顺着付款凭证、IP 记录和银行流水找到你这一端。

屏幕上是报价、昵称、完成率。对方背景你其实不了解。对方链条一旦出问题，麻烦不会只停在他那一端。

## 2. 支付风控


**支付风控**指的是：银行或支付工具（微信 / 支付宝 / 银行卡）对高频、大额、非常规对手方或命中反洗钱模型的交易主动拦截——表现为交易被标记为可疑、账户被限制部分功能、甚至银行卡整张被冻结。

大陆用户怕的多数时候不是链上失败，是支付侧异常：交易被判可疑、账户被风控、被冻结、被要求解释用途，或者临时限制一部分功能。

## 3. 问题资金


**问题资金**（也叫"黑钱"、"涉案资金"）指的是：对方打给你的人民币，其实来自已经被立案调查的其他案件（电信诈骗、赌博、洗钱、非法经营等）。一旦警方追查资金流向，**你的银行卡会被冻结、资金会被保全、本人可能需要配合调查**——即使你完全不知情，也会被牵连进流程。

这是老用户最敏感的一点。表面上你只是卖一笔 U 或买一笔 U，真正的麻烦是：进来的钱干不干净、出去的钱后面会不会带出连锁问题。

## 4. 解释成本


**解释成本**指的是：账户被临时限制、被冻结、被要求说明用途之后，你需要花在"证明这笔交易是合法的"上的时间和精力——跑银行、提供凭证、配合电话问询、等待解冻。对普通上班族来说，一张卡被冻 1–3 个月意味着工资、房贷、日常支付全部要绕路。

耗人的通常不是亏钱，是后面那堆事——解释交易、说明用途、处理限制、反复沟通。新手常常低估这一层。

## 5. 新手根本没法正确给风险定价

零点几个点的差价写在报价上，风险没有。你看到利润，对手看到流动性，看不到的是风控、链条、交易对手和各家规则的差异。

## 我的建议

- 第一次别把自己当套利者。
- 先把买入、提币、钱包、转账这几步走熟。
- 支付侧风险你能说出具体内容之后，再去谈效率和差价。

> 现实提醒 如果“问题资金”“支付风控”“账户限制”对你来说还只是几个词，那你现在该做的是小额跑路径，而不是去碰高频、压点差的玩法。
