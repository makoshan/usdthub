# 冷钱包（硬件钱包）给 USDT 长期持有人的选择指南

> Chapter 2.7 · Wallets · Last updated: 2026-04-14

钱包里的 USDT 从几百美金涨到几万美金之后，该想的不再是「下一笔怎么赚」，是「这笔钱怎么不归零」。手册前面讲过为什么不要把大额资产放在交易所（平台跑路、账号被风控、KYC 问题）；自托管热钱包（imToken / TokenPocket / 手机上的 MetaMask）也不是终点，它们还是实时联网的。真的打算长期囤 USDT / BTC / ETH 的人，下一步是冷钱包（硬件钱包）。

这篇整理自一支中文视频，回答三个问题：中国用户要不要买冷钱包、买哪一款、从哪里买。没有邀请码，没有返佣；原视频里嵌入的推广链接和 Telegram 私信返现也没搬过来。

> TL;DR ① 冷钱包就是一块离线保存私钥的硬件，给链上资产多加一层保险柜。② 中国大陆用户：买 OneKey，国内直发，不过海关。③ 国外用户：Trezor 或 OneKey，都开源。④ Ledger 固件闭源，本文不推荐拿来做大额长期冷存。⑤ 冷钱包只解决「私钥被网络偷走」，不解决「你自己被骗去签授权」。Drainer 钓鱼和假客服照样能把硬件钱包里的钱骗走。

## 一、什么是冷钱包（以及它到底防什么）

先回到区块链钱包的本质：谁拿到私钥 / 助记词，谁就能签名转走钱。不是谁「登录」，也不是谁「有账号」，是谁能签名。

热钱包（imToken、MetaMask、TokenPocket 这类手机或浏览器钱包）有一个结构性弱点：私钥存在联网设备里。钱包 App 本身做得再好，只要手机或电脑中了木马、剪贴板被劫持、浏览器装了恶意插件，私钥理论上就可能被远程拿到。

冷钱包把这部分切开：私钥的生成和签名都在一块离线的硬件里完成，电脑或手机只负责把未签名的交易发过去、把签好的交易拿回来广播。接入的电脑就算中毒，私钥也不会被带出来。

| 对比 | 热钱包 | 冷钱包 |
| --- | --- | --- |
| 打个比方 | 手机里的支付宝 | 银行保险柜 |
| 私钥储存 | 联网设备 | 离线硬件 |
| 用的时候 | 随时能转 | 要拿出来、插上、按确认 |
| 被黑客远程偷走 | 理论上可能 | 几乎不可能 |
| 被社工 / 钓鱼骗去签 | 会 | **一样会** |

最后一行特别要注意：冷钱包不防你自己被骗。你点开一个 Drainer 页面，硬件钱包弹出一条你看不懂的授权，你按了「确认」，钱照样走。手册里反复讲的那句话：地址、私钥、授权，这三件事各怕各的，别混在一起。

### 1.1 冷钱包会不会坏

会。硬件会坏，公司也可能倒闭。但只要助记词在，这些都不影响你的资产。冷钱包只是一个存私钥的容器，助记词才是私钥的人类可读版本。硬件坏了，买一个新的（任何品牌都行，只要支持同一套 BIP39），恢复助记词，钱全在。

所以真正的单点风险不是硬件，是那张助记词纸。

### 1.2 助记词纸也会坏——金属备份的意义

视频里提了一个容易被忽视的点：纸本抄写助记词，遇到火、水、搬家、家人误丢，那「理论上永存的 12 / 24 个词」说没就没。真打算用冷钱包长期囤大额资产，配一套金属助记词板（OneKey KeyTag、Trezor Keep Metal、CryptoSteel Capsule 这类不锈钢或钛合金刻字备份）值得。几十美金，换掉一个概率不低的事故。

## 二、市面上的主流冷钱包

视频只重点讨论三个主流品牌，其他小众品牌归到「等等」。这里沿用同样的分类。不是说别的品牌不好，冷钱包是押身家的设备，选一家存活时间够长、透明度够高的团队最省心。

| 品牌 | 成立 | 总部 | 典型产品 | 本文立场 |
| --- | --- | --- | --- | --- |
| **Ledger** | 2014 | 巴黎（法国） | Nano S Plus / Nano X / Stax | 不作为大额长期冷存推荐 |
| **Trezor** | 2013 | 布拉格（捷克） | Model One / Safe 3 / Safe 5 | 推荐（开源，老牌） |
| **OneKey** | 2018 | 新加坡（生产在中国） | Classic 1s / Pro / Touch | 中国用户首选（开源 + 国内直发） |

其他品牌（SafePal、imKey、Keystone、CoolWallet、Tangem、Ellipal 等）并不是不能用。但如果你刚入门，只想做一个「不必重新研究」的决定，从上面三家里挑一个基本不会错。

## 三、评价冷钱包的 4 个维度

下面是视频作者的主观标准。不是技术论文级别的完整分析，做购买决定的骨架足够清楚。这 4 条合起来一筛，剩下的选项不多。

### 3.1 代码是否开源

代码全公开，任何人能审计，不能选择性开源。这是看硬件钱包「愿不愿意被查」最硬的一条。

- Ledger：固件闭源（Ledger Recover 那次争议之后质疑更多）
- Trezor：开源
- OneKey：开源

主观但重要：长期托管 5 位数美金以上的资产，第一关就把不愿意开源审计的团队排掉，合理。

### 3.2 是否经过压力测试（真实事故）

「没出过事」不等于安全，也可能只是「还没被盯上」。更关键的是：出事之后用户资产有没有被拿走。

| 品牌 | 事故简述 | 资产是否被盗 |
| --- | --- | --- |
| Ledger | 2020 客户信息泄露（约 27 万用户地址邮箱手机号外流）；后续出现大量针对性钓鱼 | 信息泄露导致**社工攻击**，间接损失存在；私钥本身未从设备外流 |
| Ledger Connect Kit | 2023-12 前端库被注入恶意代码，部分 dApp 用户中 Drainer，报道损失约 ~$600k | 被盗发生在**dApp 前端**层，不是硬件本身 |
| Trezor | 2022 邮件服务商 MailChimp 被入侵，用户邮箱泄露 | 无直接资产损失 |
| OneKey | 截至本文写作无公开的用户资产被盗事故 | —— |

三家公司的私钥都没有从硬件里被网络端拿走。这反过来印证了冷钱包的核心假设：厂商本身既不保存、也访问不到你的私钥。但 Ledger 那两起事件提示了另一层：一家不开源、又愿意设计「让厂商服务器介入备份」方案的公司，风险敞口比纯硬件制造商大。

### 3.3 冷钱包是否是主营业务

道理很直白：公司会不会把注意力持续放在这条产品线上。Ledger、Trezor、OneKey 三家都是硬件钱包主业。那种「顺手做个硬件钱包」的综合品牌，长期维护、固件更新、事故响应的优先级往往低得多。

### 3.4 是否对中国大陆用户提供服务

冷钱包在中国是合法合规的硬件，性质跟你买手机、买 Kindle 一样。但有些品牌出于自己的合规或商务判断，明确不服务中国大陆用户：官网下单会被退单，出了问题找客服也会被一句「我们不服务大陆用户」挡回来。视频作者对这类品牌的建议是别用。不是产品不好，是你作为用户的保障太弱。

Ledger、Trezor、OneKey 目前都服务中国用户。

## 四、谁该买冷钱包

### 4.1 按资产规模

| 加密资产总额（USDT 等值） | 建议 |
| --- | --- |
| &lt; $10,000 | 热钱包 + 交易所分散存放即可，现阶段重点是学会**安全操作习惯**（参见手册 [怎么选钱包](./how-to-choose-a-wallet.md) / [怎么发送 USDT](./how-to-send-usdt.md)） |
| $10,000 – $30,000 | 配一个基础款冷钱包，把长期囤的部分搬过去 |
| &gt; $30,000 | 强烈建议配。$80–100 的硬件 / 占总资产 0.3%，换来断网存储加物理签名二次确认，合算 |

简单算一下：$80 / $30,000 ≈ 0.27%。用本金的 0.27% 给剩下 99.7% 上一道物理锁。

### 4.2 按使用场景

下面任何一种对得上，就该配冷钱包：

- 分散存储：不想把大额全压在一家交易所或一个热钱包里。
- 链上理财 / 质押：Aave、Compound、TRON 上的 JustLend、节点质押这些——授权都要硬件按一下，Drainer 很难骗你全盘授权。
- 断网囤币：BTC / ETH / USDT 长期持有，平时不动。
- 节点质押 / 验证人：一个长期只做一件事的地址，放冷钱包最合适。
- 链上交易 + 打新 + 冲土狗：玩耍资金单独放一个冷钱包地址，亏了也不会炸到囤币地址。

手册里 [怎么选钱包](./how-to-choose-a-wallet.md) 讲的是热钱包之间怎么选；这篇接在后面：确认自己进入了「要给大额加保险柜」的阶段，才需要冷钱包。

## 五、中国用户的具体购买建议

### 5.1 在中国大陆：优先 OneKey

理由不是品牌偏好，是物流：

- OneKey 在中国生产，国内下单从国内仓直发，不过海关。
- Ledger / Trezor 是欧洲公司，海外下单到中国要走海关。
- 冷钱包最怕的供应链攻击是运输途中被拆包、植入恶意固件或预设助记词。硬件在运输里被 X 光、被开箱检查、在中转仓停一段时间，风险就留下了。

中国海关对加密相关硬件没有明文禁运，但不确定性本身就是风险。冷钱包这种「被动过手脚你可能永远不知道」的设备，能走官网 → 国内仓 → 直发就尽量走。

入门型号：OneKey Classic 1s，约 $99，新手够用。

### 5.2 在国外：Trezor 或 OneKey 二选一

人在海外，官网直邮更可控，Trezor 和 OneKey 都合理：

- Trezor Safe 3 / Safe 5：老牌、开源、事故响应透明。
- OneKey Pro / Touch：界面更现代，EVM / TRON / BTC 多链一体。

视频作者自己同时用两家分散。不是担心单一品牌跑路，就是心理上的二次分散。对大多数人来说，选一家就够了。

### 5.3 关于 Ledger

不是说不能用。你手上已经有 Ledger，或者只打算放一点钱到「交互钱包」里玩 DeFi，Ledger 能用。但如果目标是「这家公司 10 年后还在、我敢把大额交给它的硬件」，闭源加 Recover 方案的历史让很多人把 Ledger 降到次级选项。

### 5.4 购买渠道的硬规矩

不管买哪家，只从官网下单。

- 可以：官网 ledger.com / trezor.io / onekey.so
- 不要：淘宝 / 闲鱼 / 亚马逊第三方卖家 / 任何「二手」「代购」「拆封测试」的硬件钱包

已经发生过的真实事故：二手 Ledger Nano 被卖家预设好助记词，买家充值后币瞬间被转走。冷钱包是一件你下一代都可能还在用的设备，采购环节省下那几十美金不值。

## 六、收到硬件之后要做的几件事

不管最终选了哪家，包装到手的那一刻要检查：

1. 包装完好：没有拆封痕迹，安全封条没断。
2. 不用厂家预置的 PIN / 助记词：自己走一遍「新建钱包」，亲眼看着设备生成助记词。
3. 测试恢复：在某条链上放极小金额（比如 $10 USDT），完整跑一遍「擦除设备 → 用助记词恢复 → 地址一致」，确认助记词抄写无误。
4. 金属备份：搬大额资产之前，把助记词刻到金属板上（OneKey KeyTag、Trezor Keep Metal 都行）。
5. 助记词和硬件分开放：硬件在家，助记词在银行保险箱，或者反过来。不要拍照、不要上传网盘、不要发微信 / Telegram 给自己。

## 七、硬件钱包不解决的事

最后一节专门讲冷钱包不是万能的。很多新手买了之后就掉以轻心，原因就在这里。

- Drainer 钓鱼网站：你连上钱包，页面让你「签名验证」，你按硬件上的「确认」，资产被授权转走。硬件不认语义，只认你按没按确认。
- 假客服：「我是 OneKey 客服，您的钱包有异常，请把助记词发给我们验证」。任何伸手要助记词的客服都是骗子。
- 朋友借用：别人拿走解锁后的硬件插电脑、或者在你面前转账时记下你的 PIN——冷钱包防网络，不防现场。
- 授权合约没撤销：之前授权过的 DeFi 合约，就算你换到硬件钱包地址，授权还在的话对方合约照样能转走对应代币。定期用 revoke.cash 清。

冷钱包把攻击面从「网络全世界」缩到「你自己 + 你身边的人 + 你授权过的合约」。剩下的部分还是得靠手册里反复讲的操作习惯。
