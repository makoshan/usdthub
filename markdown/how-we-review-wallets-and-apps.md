# How We Review Wallets and Apps

> Trust page · Last updated: 2026-04-14

这个手册是第三方教程站，不是交易平台，也不是谁给钱多就写谁的榜单。我们评测钱包和应用，看的是：新手看得懂吗、能不能安全把事做完、放到 USDT 的实际用法里合不合适。

> 短版结论 我们不按 payout 大小排推荐顺序。谁更好懂、更适合新手、更适合当前这个任务，就排在前面。

## 我们看什么

- 新手友好度：安装、创建、备份、收发，流程清不清楚。
- 自托管支持：控制权是不是真的在用户手里，恢复流程写没写清。
- 网络支持：TRON / TRC20 / USDT 这条路径走得顺不顺。
- 转账和费用提示：错链、手续费、资源不足的提醒够不够直白。
- 产品成熟度：口碑、更新频率、有没有长期维护。

## 推荐会以什么方式出现

- editor’s pick
- beginner-friendly option
- easiest path for a specific task

推荐会放在具体场景里，不会把某一个钱包写成全站唯一主角。

## 我们不做什么

- 不作为交易平台
- 不提供投资建议
- 不代用户买 USDT 或代管资产
- 不承诺某一个钱包适合所有人
