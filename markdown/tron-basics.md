# TRON 是什么：TRX、超级代表、3 秒出块

> TRON Note · Last updated: 2026-04-14

这篇只回答一个问题：你拿着 TRC20 USDT，为什么 TRON 链能让你 3 秒内确认、几乎零手续费转出去？

> TL;DR TRX 是 TRON 的原生代币，相当于链的「油」。每 3 秒出一个区块，由 27 个超级代表（SR）轮流打包，SR 由持币人投票选出。发 USDT 时烧的「能量」本质是 TRX 的链上消耗。新手记住：TRX 付手续费、SR 出块、投票可选。

## TRX 四种用途

1. **付交易费** —— 能量 / 带宽不够时，链自动烧 TRX 补。
2. **质押换资源** —— 冻 TRX 得每日能量配额。
3. **投票选 SR** —— 1 TRX 质押 = 1 票，得分红。
4. **部署 / 调用合约** —— 开发者付 TRX。

新手钱包里常备 20–50 TRX 周转即可。

## 27 个超级代表 + 3 秒出块

- 所有 TRX 持有者都可以投票。
- 每 6 小时统计一次，得票前 27 名当选 SR，第 28–127 名为超级合伙人。
- SR 轮流出块，每 3 秒 1 个，一天约 28,800 块。
- 投票更新时间：UTC 0:00 / 6:00 / 12:00 / 18:00。

这一节的数字可以对照 TRON 官方资源模型和 TronScan 的链上数据，不用再背一套钱包品牌叙事。

## 跟你发 USDT 的关系

- 3 秒出块 → TRC20 USDT 转账秒确认。
- USDT 是合约 → 调用要烧能量 → 没质押就要烧 TRX → 钱包里必须有 TRX。
- 长期高频用户质押几千 TRX 换日配额，单笔边际成本趋近 0。

## 投票跟新手有关系吗

不强制。如果你已经为能量质押了 TRX，顺手投头部 SR 能多 3%–6% 年化，没有额外锁仓成本。
