# TRON 能量计算器：质押多少 TRX，够转几笔 USDT

> Chapter 3.5 · Tool · 当前网络快照口径

我先说结论：这页不是讲“TRON 是不是便宜”，而是直接帮你算——你质押多少 TRX，大概能拿到多少能量，够不够转 1 笔 / N 笔 TRC20 USDT，不够还差多少。

> 在线工具页：<https://makoshan.github.io/usdthub/tron-energy-calculator.html>

## 这页解决什么问题

**这页**专门解决四件事：

1. 质押多少 TRX
2. 大概能拿到多少能量
3. 够不够转 1 笔 / N 笔 TRC20 USDT
4. 不够还差多少

## 我用的估算口径

**这页统一采用的口径**是：

- 老地址：约 65,000 能量 / 笔
- 新地址：约 130,000 能量 / 笔
- 当前网络快照：按 TRON 当前总能量 / 总质押权重反推
- 保守模式：在当前快照基础上再预留安全边际

## 你真正该记住的一句

**“质押免费”只在覆盖范围内成立。** 更准确的说法是：你先拿本金换每天刷新的能量额度，额度内免费；超出额度，还是要租或烧。
