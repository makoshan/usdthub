# 用 USDT 做跨境小额结算前要先确认什么

> Use Case · Cross-border Payments

USDT 适合做跨境小额结算，因为在某些场景里它的路径更直接。但“能不能结算”远不止链上这一步。

> TL;DR 跨境小额结算前先确认四件事：双方说的是不是同一条链、对方真能收这类资产、后续怎么落地用、当地规则和服务条款是否允许。

## 最该先确认的 4 件事

- 双方说的是不是同一条网络。
- 对方收到后是不是真的会用。
- 这笔钱怎么落地。
- 合规和平台规则是否允许。

## 最常见误区

- 你觉得“发到了就结束”，对方觉得“我还没法用”。
- 双方都说收 U，但没说哪条链。
- 只看链上动作，忽略地区和服务规则。

> 提醒 跨境支付的现实限制往往不在区块链本身，而在落地使用、当地规则、收款方处理能力和平台条款上。
