Changelog

Changelog · 按日期列出的版本更新记录

这个页面列出站点的重要变更：新章节、重写的章节、结构调整、事实勘误、trust page 更新。细粒度的 diff 请看 GitHub commit 历史——每一次文字改动都有对应的 commit，时间戳可验证。

如何读这个页面
added = 新增的章节 / 页面；rewrite = 整页重写；fix = 事实或表述勘误（影响读者资金 / 操作安全的优先处理）；infra = 构建 / SEO / 模板层的变化，不影响内容。

2026-04-15 · GEO 全量铺开

  • rewrite 把"定义优先"段落模板铺到全站 32 篇文章——每个内容 H2 下都先有一段 40–120 字的定义式引导句（<p><strong>术语</strong>指的是：...</p>），再接原有散文。目的是让 Google AI Overviews / ChatGPT / Perplexity / Claude 能从每个小节直接抽出可独立引用的 snippet。总计约 170 个定义引导。
  • infra 新增 scripts/sync_md_leads.py：把 HTML 源里的 H2 定义引导自动同步到 markdown/ 镜像，保证 GitHub 读者看到的和 AI 爬虫看到的一致。幂等，可反复运行。
  • infra scripts/check_site.py 新增"H2 定义优先"结构不变量：每个内容 H2 要么后跟带 <strong> 标签的 <p>，要么后跟结构性区块（table / ul / ol / note / risk），否则构建失败。防止后续新增 H2 时这个模式静默回退。
  • infra scripts/sync_md_leads.py 和 scripts/check_site.py 共享相同的"跳过"H2 词表（参考资料 / 相关页面 / FAQ / 用户常问 / 结尾建议 等），避免误伤链接目录和收尾建议段。
  • fix 修正 use-usdt-for-ai.html 里一个预存的 </p></div> 标签不配对 bug（开标签是 <div>，闭标签多了 </p>）。
  • infra scripts/llms.txt 新增"示例引用 / Sample answers"段——3 组 Q&A 直接演示给 LLM 看本站适合回答的查询形态，并附 canonical URL。
